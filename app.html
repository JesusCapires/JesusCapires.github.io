App Movil

Aplicación móvil para inventario de dispositivos

Desarrollo de una aplicación móvil multiplataforma dedicada al control de inventario de dispositivos tecnológicos. Su objetivo es agilizar la creación de inventarios digitales de manera eficiente y rápida. Además, proporciona una administración y organización efectiva para los usuarios de estos dispositivos.

¿Qué se usó en el desarrollo e implementación?

  • Lenguaje Dart con framework flutter
  • Pruebas de caja negra y caja blanca
  • Base de datos SQLite
  • Metodología Scrum
  • IDE Android Studio

Funciones

  • Gestión de cuentas
  • Registro de dispositivos
  • Exportación e importación de registros .xlsx
  • Exportar registros en archivo .sql
  • Resguardos PDF
  • Generación y scanner de códigos QR
  • Filtro de datos
